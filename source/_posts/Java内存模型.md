## HUE
本地源码构建安装：

```
cd <hue_dir_path>
export LDFLAGS=-L/usr/local/opt/openssl/lib
export CPPFLAGS=-I/usr/local/opt/openssl/include
make clean
make apps
./build/env/bin/supervisor
登录默认用户名密码：admin/admin
```
